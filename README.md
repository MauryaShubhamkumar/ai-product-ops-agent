# AI Product Ops Agent

An AI-powered agent for automating Product Operations tasks.

## Project Structure

- `data/`: Contains dataset inputs and source files.
- `prompts/`: System, user, and agent prompts templates.
- `src/`: Source code for the product ops agent.
- `report/`: Generated execution reports and analysis files.
- `screenshots/`: Visual logs, screenshots, and diagrams.
- `output/`: Processed outputs and files.
- `tests/`: Unit and integration test suites.
